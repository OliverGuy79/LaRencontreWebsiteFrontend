# LaRencontreWebsiteFrontend
Frontend Repo for the Website of the Chruch LaRencontre

## Branches
- `main` : Production branch
- `develop` : Development branch

## Commandes générales

- Cloner le projet : `git clone https://github.com/OliverGuy79/LRWebsiteFrontend.git`
- Se positionner sur une branche : `git checkout <nom_de_la_branche>`
- Installer les dépendances en général : `npm install`
- Installer la dépendance `vite` : `npm install -D vite`
- Lancer le serveur de développement : `npm run dev`
- Lancer le serveur de production : `npm run build`
- Lancer le serveur de production : `npm run preview`
- Créer une branche : `git checkout -b <nom_de_la_branche>`
